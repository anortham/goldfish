## Fixed 3 remaining audit bugs (TDD)

Applied fixes for the 3 bugs identified during the comprehensive project audit, all with tests written first:

### Bug 2: Plan handler drops `tags` and `id` on save
- **Root cause:** `src/handlers/plan.ts:17` only destructured `title`, `content`, `activate` — ignored `id` and `tags`
- **Fix:** Added `id` and `tags` to destructuring, forward them conditionally to `savePlan()`
- **Tests:** 2 new tests in `handlers.test.ts` (forwards tags, forwards custom id)

### Bug 3: `getDateRange` to-only bug
- **Root cause:** `src/recall.ts:113` calculated `from` as 7 days before `now` instead of 7 days before the `to` date
- **Fix:** Parse `to` as `Date`, calculate `weekBefore` relative to it
- **Tests:** 1 new test in `recall.test.ts`

### Bug 4: `savePlan` non-atomic write + missing trailing newline
- **Root cause:** `formatPlanFile` didn't end with `\n`; `savePlan` used `writeFile` with `wx` flag (not crash-safe)
- **Fix:** Added trailing `\n` to `formatPlanFile`, converted `savePlan` to write-temp-then-rename pattern, updated `parsePlanFile` to strip trailing newline from content
- **Tests:** 3 new tests in `plans.test.ts` (trailing newline formatting, atomic write, file trailing newline)
