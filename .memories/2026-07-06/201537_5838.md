## Released and verified Goldfish 7.4.2

### WHAT
Published Goldfish `v7.4.2` after discovering that the initially created `v7.4.1` source archive failed install smoke. `v7.4.2` is the verified latest release.

### WHY
The release flow required live GitHub verification plus downloaded artifact smoke. `v7.4.1` included the Windows registry and stuck `roots/list` fixes, but its GitHub-generated source tarball failed `bun install --frozen-lockfile` because package `prepare` assumed a git checkout. `v7.4.2` supersedes it with the prepare-script fix.

### HOW
- Pushed `main` and annotated tag `v7.4.2`.
- Created GitHub release `v7.4.2` with notes covering the source archive fix and the superseded `v7.4.1` registry/roots fixes.
- Verified live release metadata: non-draft, non-prerelease, Latest, tag dereferences to release commit `3f46ee794891ecf6da45d81ab97d923411851725`.
- Downloaded `https://github.com/anortham/goldfish/archive/refs/tags/v7.4.2.tar.gz`, checked version strings, installed dependencies from the archive, and ran build/typecheck/server tests.

### IMPACT
Users should install/use `v7.4.2`, not `v7.4.1`. The latest release includes the duplicate Windows registry dedup fix, bounded MCP `roots/list` recovery, Bun build target correction, and source archive install safety.
