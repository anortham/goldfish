## Fixed all cross-platform test failures (Windows)

231 tests passing, 0 failures — project now works on Windows.

### What was broken
- **5 errors**: Missing `node_modules` (yaml, fuse.js not installed)
- **3 test failures**: Hardcoded Unix path assumptions in test assertions

### Root causes & fixes

- **`tests/workspace.test.ts`**: Hardcoded forward-slash path expectations (`/some/path/.memories`) fail when `path.join()` returns backslashes on Windows. Fixed by using `join()` in assertions too. Also replaced `/tmp/...` with `join(tmpdir(), ...)`.
- **`tests/registry.test.ts`**: `.startsWith('/')` check fails on Windows where paths start with drive letters (`C:\`). Replaced with `isAbsolute()`.
- **`tests/checkpoints.test.ts`**: Auto-registration test used `/tmp/...` as tempDir, but `resolve('/tmp/...')` becomes `C:\tmp\...` on Windows, so `p.path === tempDir` comparison failed. Fixed with `join(tmpdir(), ...)`.
